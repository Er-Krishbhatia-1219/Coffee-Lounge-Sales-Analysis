☕ Coffee Lounge Sales Analysis Dashboard
📌 Project Overview

This project presents a comprehensive sales analysis dashboard for a coffee lounge business using Microsoft Excel. The dashboard provides business insights related to sales performance, customer footfall, product demand, and operational trends.

The aim of this project is to transform raw transactional data into meaningful business intelligence that helps stakeholders make data-driven decisions.
